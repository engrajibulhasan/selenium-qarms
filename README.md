# selenium-qarms

## How to setup Selenium in ubuntu

```bash
sudo apt update
sudo apt install python3-full python3-venv -y
```

Then

```bash
python3 -m venv selenium-env
```

Then

```bash
source selenium-env/bin/activate
```


## How to start with slenium env

python test_qarms_sso_generate_html.py
